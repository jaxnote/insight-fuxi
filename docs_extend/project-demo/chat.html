🔍 核心洞察

上月 GMV 环比下降 12%，从 8,560 万降至 7,533 万。主要原因：

  • 3C 数码品类转化率下降 23%（贡献度 58%）— 3C 品类 GMV 从 1,200 万降至 920 万，是最大拖累因素
  • 华北地区 UV 下降 15%（贡献度 27%）— 华北 UV 从 50 万降至 42.5 万
  • 其他品类和地区表现基本稳定，波动在 ±5% 以内

💡 建议

  • 建议关注 3C 品类的商品详情页体验，排查是否有页面改版或加载性能问题
  • 排查华北地区的流量投放策略变化，确认是否有投放预算调整
